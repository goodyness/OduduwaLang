# OduduwaLang

**OduduwaLang** is a serious, open-source, Yoruba-first programming language. It is designed to allow developers to build real-world, production-capable software using the rich grammar and vocabulary of the Yoruba language.

## 🌟 Vision
Programming should be accessible in one's native tongue without sacrificing technical depth or power. OduduwaLang provides a culturally rooted yet technically sophisticated alternative for the global Yoruba-speaking community.

## 🛠 How it works
OduduwaLang builds on the Python ecosystem. It features:
- **Yoruba Syntax**: Keywords like `ise`, `ti`, `fún`, and `tẹjáde`.
- **Diacritic-Awareness**: Tone marks are optional in execution but preserved for documentation.
- **Python Transpilation**: Runs anywhere Python runs.
- **Yoruba Error Reporting**: Understand bugs in your own language.

## 🚀 Roadmap
- **Phase 1**: Core Grammar & Basic Transpiler (Variables, Functions, If-statements).
- **Phase 2**: Advanced Logic (Loops, Data Structures) & Standard Library.
- **Phase 3**: Localized Developer Experience (CLI, REPL, Yoruba Stack Traces).
- **Phase 4**: Community Ecosystem (VS Code Extension, Full Documentation).
